# Learning Resources

## Useful Git & GitHub Links
- [GitHub Guides](https://guides.github.com/) - Mastering Issues, GitHub Pages, and more.
- [Git Documentation](https://git-scm.com/doc) - Official documentation and book.
- [Atlassian Git Tutorial](https://www.atlassian.com/git/tutorials) - Comprehensive tutorials for all levels.
- [GitHub Skills](https://skills.github.com/) - Interactive courses to learn GitHub.

## Practice Tools
- [Learn Git Branching](https://learngitbranching.js.org/) - An interactive visualization of Git commands.
- [Oh My Git!](https://ohmygit.org/) - An open-source game about learning Git.

## Checklist for Beginners
- [ ] Initialize a repository
- [ ] Clone a remote repository
- [ ] Create a feature branch
- [ ] Add and commit changes
- [ ] Push to GitHub
- [ ] Open a Pull Request
- [ ] Merge a Pull Request
